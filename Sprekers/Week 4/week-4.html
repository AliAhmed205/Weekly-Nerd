Nils Binder, a versatile expert in communication design, product development, and web development, shared some engaging insights during his Weekly Nerd session from his time at 9Elements in Dortmund.

Units I Know:

  • px: Pixels - Absolute unit, often used for precise dimensions.
  • %: Percentages - Relative unit, related to the size of the parent container.
  • rem / em: Root em - Relative units, based on the size of the root (html) element.
  • vw / vh: Viewport width / height - Relative units, based on the size of the browser viewport.
  • dvw / dvh: Dynamic viewport width / height - Relative units, dependent on the size of the browser viewport.

Units I Don't Know:

  • vi / vb: Viewport inline direction / viewport-percentage based on standard viewport size of the browser.
  • lvh / lvb / lvw / lvi: Largest possible dimension in block direction - Based on the largest possible dimension in block direction.
  • svh / svb / svw / svi: Smallest possible dimension in block direction - Based on the smallest possible dimension in block direction.
  • dvb / dvi: Depending on device viewport - Based on the viewport of the device.
  • lvmin / svmin / dvmin: Viewport minimum - Based on the minimum of the viewport.

In addition to the wrapper element, Nils also discussed the optional column technique, a grid-based layout concept he enjoys using. By defining columns with the "fr" unit, he can create a flexible layout that adapts to different screen sizes. This allows him to optimize the layout of his websites and highlight key content while considering responsiveness and user experience.
